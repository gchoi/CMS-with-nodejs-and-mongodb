Mac을 켜자마자 보게 되는 데스크탑으로부터
날마다 사용하는 응용 프로그램까지 모든 것이 간결하고
직관적으로 설계되었습니다.
대단한 기능을 손쉽게 사용할 수 있게 하려면
정말 우수한 기술이 필요합니다.
특징
철저한 보안 탑재.
OS X는 바이러스와 그 밖의 악성 응용 프로그램을 차단하도록 디자인되어 있습니다. 모든 Mac에는 보안 설정이 내장되어 있으므로, 안전을 유지하기 위해 복잡한 설정을 바꿀 필요가 없습니다.
호완성을 위해 구축되었습니다.
다재다능하고 강력한 OS X은 Windows 네트워크는 물론, 어떤 환경에서든지 Mac을 사용할 수 있게 해줍니다. Windows PC에 있는 파일, 사진 및 기타 데이터를 전송하고 싶다고요?
컴퓨터의 사용이 혁신적으로
쉬워졌습니다.
OS X는 Mac의 기능을 일반적으로 접근하는데 어려움이 있는 사용자들도 충분히 사용할 수 있도록 다른 운영체제에서는 절대 찾아볼 수 없고 값을 매길 수 없는 다양한 기능을 지원합니다.
특징
다운로드
MacBook Air의 놀라운 반응 속도는 flash 스토리지 중심의 설계 덕분입니다. 이렇게 설계된 MacBook Air는 기존의 다른 노트북보다 훨씬 더 가볍고 휴대성이 뛰어날 뿐만 아니라, 더욱 빠른 데이터 액세스와 언제나 즉시 켜지는 빠른 속도를 제공합니다. 최대 512GB까지 제공되는 차세대 flash 스토리지는 이전 세대 MacBook Air보다 빠르며, 5400rpm인 기존 노트북의 하드 드라이브보다 최고 4배의 속도를 제공합니다.2언제라도 MacBook Air를 열면 몇 초만에 바로 켜집니다.
